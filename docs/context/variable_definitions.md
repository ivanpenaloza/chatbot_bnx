
| Variable          | Definition                                                                                                                                                                                                                                                                 |
| :---------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| etiqueta_grupo    | Clasificación que segmenta a los clientes en dos grupos principales: 'potenciales' (clientes prospecto a ciertos productos de crédito) y 'mailbase' (clientes a los que se les hará llegar la oferta de un producto de crédito).                                                   |
| producto          | Indicador que especifica el tipo de producto de crédito (ej. CLI, DC, TCC).                                                                                                                                                                                                  |
| toques            | Número acumulado de intentos de oferta realizados al cliente antes de la aceptación del producto.                                                                                                                                                                             |
| overlap_inicial   | Producto o conjunto de productos de crédito con las que el cliente inicia el proceso de Overlap.                                                                                                                                                                              |
| asignacion_final  | Producto o conjunto de productos de crédito definitivos asignados al cliente después del proceso de Overlap.                                                                                                                                                                  |
| ds_testlab        | Tests a los que pertenece el cliente.                                                                                                                                                                                                                                      |
| escenario         | Estrategia o regla de negocio bajo la cual se definieron las ofertas de productos para el cliente.                                                                                                                                                                           |
| conteo            | Número total de clientes que comparten las mismas características (mismo producto, escenario, test, etc.).                                                                                                                                                                  |
| linea_ofrecida    | Suma total del monto de líneas de crédito ofrecidas a todos los clientes dentro de las mismas características.                                                                                                                                                              |
| npv               | Suma del Valor Presente Neto (NPV) proyectado para los clientes con las mismas características.                                                                                                                                                                             |
| rentabilidad      | Suma de la rentabilidad proyectada para los clientes con las mismas características.                                                                                                                                                                                       |
| rr                | Suma del RR proyectado para los clientes con las mismas características.                                                                                                                                                                                                   |
| campania          | Indicador de campaña.                                                                                                                                                                                                                                                     |
| flag_declinado    | Indicador binario que señala si se ofreció el producto o no al cliente.                                                                                                                                                                                                     |
| causa_no_asignacion | Descripción que especifica el motivo por el cual no se realizó la oferta del producto al cliente (ej: capacidad de pago).                                                                                                                                                   |
